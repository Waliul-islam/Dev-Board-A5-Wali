Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. I will mention mostly used 5 ways. They are following:

  1. getElementByID()
  2. getElementByClassName()
  3. getElementByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText - এই প্রোপার্টি ব্যবহার করে কোন Element এর মধ্যে টেক্সট যোগ করা যায় বা তাকে এক্সেস করা যায়।

innerHTML - এই প্রোপার্টি ব্যবহার করে কোন Element এর মধ্যে ট্যাগসহ HTML যোগ করা যায়।

textContent - এইটা অনেকটা এর মতই তবে, innerText HTML & CSS এর ভ্যালু অনুযায়ী টেক্সট দেখায় কিন্তু textContent এ এর কোন প্রভাব থাকেনা। অর্থাৎ কোন ইলিমেন্টে যদি display:hidden, none সেট করা থাকে তা ও textContent দ্বারা দেখা যায়, যা innerText দ্বারা করা সম্ভব হয়না।

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি সিস্টেম যেখানে আমরা প্যারেন্ট এলিমেন্টে ইভেন্ট লিসেনার অ্যাড করি, এবং চাইল্ড এলিমেন্টে সেই ইভেন্ট স্বয়ংক্রিয়ভাবে কাজ করে। এইটা ব্যবহারের একটা সুবিধা হলো নতুন এলিমেন্ট যোগ করলেও ‍অটোমেটিক ওইটাতেও কাজ করে।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো সেই প্রসেস যেখানে কোনো ইভেন্ট ট্রিগার হলে, তা সবচেয়ে ভেতরের এলিমেন্ট থেকে বাইরের এলিমেন্ট পর্যন্ত "বাবল" করে উপরে উঠে। এইটা অনেকটা Event Delegation এর বিপরীত প্রক্রিয়া।

Question-5: How do you create, add, and remove elements using JavaScript?

Create Element: document.createElemet() ব্যবহার করে নতুন ইলিমেন্ট তৈরী করা যায়।

Add Element: .appendchild() এর মাধ্যমে নতুন তৈরীকৃত ইলিমেন্ট এ্যাড করা যায়।

Remove Element: .removeChild() এর মাধ্যমে নতুন তৈরীকৃত ইলিমেন্ট মুছে ফেলা যায়।
